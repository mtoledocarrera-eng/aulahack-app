# REGLAS PEDAGÓGICAS — 360Hacks

## Adecuaciones DUA Obligatorias
1. **Toda** respuesta de IA que genere contenido pedagógico DEBE incluir adecuaciones DUA en tres principios:
   - **Representación**: Múltiples formas de presentar la información (visual, auditiva, kinestésica)
   - **Acción y Expresión**: Opciones diferenciadas para que el estudiante demuestre aprendizaje
   - **Compromiso**: Estrategias para mantener motivación y autorregulación

## Validación de Estructura de Clase
2. Toda planificación DEBE validarse contra la estructura de 5 pasos antes de entregarse al docente:
   - Inicio (≥5 min)
   - Desarrollo (≥25 min)
   - Cierre (≥5 min)
   - Evaluación (criterios explícitos)
   - Adecuaciones DUA (≥1 por principio)

## Objetivos de Aprendizaje (OAs)
3. **PROHIBIDO** inventar OAs. El sistema DEBE:
   - Buscar en la base RAG curricular
   - Si no encuentra → activar Flip Prompting para clarificar con el docente
   - NUNCA generar un OA que no exista en el currículum oficial Mineduc

## Decretos Aplicables
4. Referencias legales obligatorias:
   - **Decreto 67**: Evaluación, calificación y promoción
   - **Decreto 83**: Diversificación de la enseñanza (DUA)
   - **Bases Curriculares**: Según nivel y asignatura del docente

## Flip Prompting
5. Cuando el docente no especifica nivel, asignatura u OA, el sistema DEBE:
   - Generar preguntas clarificadoras (máximo 3)
   - NO asumir valores por defecto
   - Presentar opciones cuando sea posible (e.g., lista de OAs del nivel)
